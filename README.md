# Forged Wheels Showroom

Виртуальный шоу-рум кованых дисков на заказ с примеркой на фото авто.

**Рынок:** Европа, США, Китай и т.д. (НЕ Россия)

## 🛠 Технологический стек

- **Фронтенд:** HTML + CSS + JavaScript (без фреймворков)
- **AI-генерация:** fal.ai (Flux Schnell/Dev/Pro, BiRefNet, Flux Fill)
- **Локальная обработка:** Python + Pillow + rembg
- **3D:** Three.js (через CDN)
- **Дев-сервер:** Claude Preview на localhost:3456
- **Деплой:** Vercel / Netlify (планируется)

## ⚡ Быстрый старт

1. Открыть ярлык "Подбор дисков" на рабочем столе (или `http://localhost:3456`)
2. Сервер запускается через Claude Preview
3. Редактирование: `index.html` → refresh в браузере

## 📜 Главные правила проекта

1. **Всё сохранять в этой папке** — никаких файлов в /tmp, Desktop или других местах
2. **Версии перед изменениями** — копировать в `versions/` перед большими правками
3. **Качество > экономия** — fal.ai там где нужно качество, бесплатно где можно без потери
4. **Рынок не Россия** — контент/дизайн/цены под Европу/США/Китай
