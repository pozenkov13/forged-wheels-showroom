## 🎯 ГЛАВНАЯ ИДЕЯ

Превратить сайт из плоского лендинга в **объёмный 3D-шоу-рум**, как будто клиент заходит в настоящий магазин:
- Диски развешаны как на витрине
- Вместо продавца — generic макет автомобиля (без марки)
- Клиент загружает свою машину, выбирает диски с витрины
- Всё ощущается как премиум-бутик

### 1️⃣ Задача 1: Виртуальный шоу-рум с 3D-витриной

**Что делаем:**
- Секция "Showroom" с объёмным ощущением
- Диски "развешены" на виртуальных стендах/стеллажах (не плоская сетка)
- В центре — silhouette generic автомобиля (без марки — абстрактный силуэт)
- Клиент может загрузить свою машину → силуэт заменяется на его фото
- Клиент выбирает диски с витрины → они визуально накладываются на его машину

**Acceptance criteria:**
- [ ] Секция выглядит как 3D-пространство, не плоская сетка
- [ ] Generic car silhouette (НЕ BMW/Mercedes/Audi/конкретная марка)
- [ ] Upload фото машины → замена силуэта
- [ ] Hover по диску в витрине → preview на машине
- [ ] Клик по диску → "примерить" на фото клиента
- [ ] Анимации плавные, натуральные (как в шоуруме)

**Технологии:** CSS 3D transforms, Three.js для реальной 3D-сцены если нужно

**Приоритет:** 🔴 КРИТИЧНО (это ядро USP)

## 🏬 УТОЧНЕНИЕ: Стиль 3D-шоу-рума

**Решение владельца:**
- **Стиль:** Realistic 3D (фотореалистичный, не стилизованный)
- **Шоу-рум:** реалистичный, как настоящий магазин
- **Диски:** реалистичные 3D-модели (не иконки)

**Ключевая механика — "самообслуживание":**
> Как в магазине самообслуживания — клиент САМ берёт товар с полки и кладёт в корзину.

**User experience:**
1. Клиент видит витрину с дисками (как на стеллажах магазина)
2. Берёт диск (hover → lift + scale + shine)
3. **Прикладывает/"устанавливает"** его на свою машину (drag or click to apply)
4. Может снять → поставить другой
5. Покупает когда нашёл "свой"

**Почему это важно:**
- Привычное поведение (люди знают магазины самообслуживания)
- Тактильность = вовлечённость
- Чувство контроля = выше конверсия
- **Уникально для премиум-сегмента** — никто так не делает

### 🔄 ПОДХОД К РЕАЛИЗАЦИИ — ГИБРИД (решение владельца)

**Два параллельных трека:**

**Трек 1: MVP 2.5D (2-3 недели)**
- Реалистичные PNG-диски с прозрачным фоном (у нас уже есть)
- CSS 3D transforms для эффекта полок/перспективы
- Drag-and-drop на фото машины клиента
- Анимации "взял → поднёс → приложил"
- Эффекты света/тени/отражения через CSS
- **Цель:** быстро выпустить, валидировать концепцию, собрать feedback

**Трек 2: Поиск 3D-моделлера (параллельно)**
- Искать 3D-artist (Upwork, Fiverr, Behance)
- Бюджет: $100-200 за диск × 10-15 дисков = $1,000-3,000
- Примеры требуемой работы: Sketchfab premium wheels
- Срок: 3-4 недели на создание первых 5 моделей

**Трек 3: Upgrade до full 3D (через 2-3 месяца)**
- Когда MVP провалидирован + 3D-модели готовы
- Three.js сцена с реалистичным освещением
- Полный 3D drag-and-drop
- Физика и отражения

## 🎨 ДОПОЛНИТЕЛЬНАЯ МОНЕТИЗАЦИЯ: Design Studio (важно)

**Идея владельца:** отдельный раздел / "сайт в сайте" — **custom wheel design service**.

Клиент приходит к нам когда:
- **Не нашёл подходящий диск** в каталоге
- **Нет собственного дизайнера**
- **Свой дизайнер слишком дорогой**
- **Нет идеи** но хочет что-то уникальное
- **Хочет брендированный диск** (свой логотип, монограмма)

### Что предлагаем

**Услуга:** создание уникального дизайна кованого диска по запросу клиента.

**Деливериз:**
- 3D-модель диска
- Фотореалистичный рендер на его машине
- Техническое ТЗ для производства
- Готовность к заказу

### Структура блока "Design Studio"

1. **Hero секция** — "Don't see your dream wheels? We'll design them"
2. **How it works** — 4 шага (Brief → Sketches → 3D → Production)
3. **Gallery** — примеры уникальных дизайнов
4. **Pricing tiers:**
   - **Basic:** €500-800 (1 концепт, 2 правки)
   - **Standard:** €1,200-1,800 (3 концепта, 5 правок, 3D)
   - **Premium:** €2,500-4,000 (концепт + брендинг + exclusive rights)
5. **Request form** — бриф для запуска работы
6. **FAQ** по дизайн-процессу

### Монетизационная модель — гибрид с психологическими триггерами

**Две услуги:**
1. **Design only** — разработка дизайна
2. **Design + Manufacturing** — дизайн + изготовление (bundle)

### 🎁 Ключевые офферы (добавлены владельцем)

#### Оффер 1: Bundle Discount
**Если клиент делает дизайн у нас И заказывает изготовление — получает скидку.**

Варианты реализации:
- **Вариант A:** Стоимость дизайна **вычитается** из цены дисков (credit back)
- **Вариант B:** **30% скидка** на дизайн если заказывает изготовление
- **Вариант C:** **Free design** если общий заказ > €5,000

**Эффект:** клиент понимает что платит дизайн "бесплатно" → выбирает bundle → больше средний чек.

#### Оффер 2: "Money-back guarantee" — Risk-free design
**«Если ни один концепт вам не подойдёт — вы не платите ничего.»**

Как работает:
- Клиент делает бриф
- Мы создаём 2-3 концепта
- Если ничего не нравится после N итераций → полный возврат
- Гарантия убирает страх "отдам €1,200 и не понравится"

**Эффект:** снимает психологический барьер "а вдруг не получится" → больше людей начинают дизайн-процесс.

### Итоговая маркетинговая связка

```
Hero Design Studio:
"Design your dream wheels with us.
If nothing suits you — you pay nothing.
Order manufacturing with us — design is on us*"

*applies to orders over €5,000
```

🇷🇺 «Создайте диски мечты с нами. Если ничего не подойдёт — вы не платите. При заказе изготовления — дизайн за наш счёт*»

### Почему это работает

| Психологический триггер | Как работает |
|---|---|
| **Risk reversal** | Убираем страх потери → больше людей пробуют |
| **Anchoring** | €1,200-4,000 за дизайн → €3,400 за диски кажутся недорого |
| **Commitment** | Клиент вложился в дизайн → не уйдёт к конкурентам |
| **Bundle discount** | Ощущение "получил дизайн бесплатно" → больше ценности |
| **Exclusivity** | "Свой уникальный дизайн" → статус, гордость |

**Рекомендация:** комбинация — **risk-free design + bundle discount** (обе механики).

## 🎬 ДОПОЛНИТЕЛЬНАЯ МОНЕТИЗАЦИЯ: Native Ads в Tutorial-видео

**Идея владельца:** tutorial-видео "Как пользоваться сайтом" = оплачиваемое product placement для автобрендов.

### Концепция

Короткий обучающий видеоролик (15-30 сек) встраивается на главной или в onboarding flow:

1. Клиент заходит на сайт → видит CTA "Watch how it works" (15 sec)
2. В ролике **выезжает реальная машина** конкретного бренда (Porsche 911, Mercedes AMG GT, BMW M4, Audi RS6)
3. Показывается анимация: пользователь "берёт" диски с витрины → "надевает" на эту машину
4. Видео служит:
   - **Инструкцией** как пользоваться сайтом
   - **Product placement** для автобренда
   - **Демонстрацией** нашей AI-примерки

### Бизнес-модель

**Партнёрства с автобрендами:**
- Porsche / Mercedes-AMG / BMW M / Audi RS / Aston Martin / Maserati
- Договор на показ **новой/флагманской модели** в tutorial
- Ротация — разные бренды для разных клиентов (по гео, интересам)
- **Оплата за показы или фиксированный контракт** (license fee per quarter)

### Варианты монетизации

| Модель | Логика | Потенциал |
|---|---|---|
| **License fee** | Бренд платит за квартал/год показов | €5,000-30,000/квартал |
| **CPM** | Оплата за 1000 показов | €10-50 CPM |
| **Performance** | % от продаж бренда через referral | партнёрский доход |
| **Trade** | Бесплатное размещение в обмен на контент/visibility | взаимный PR |

### Примеры сценариев

**Сценарий 1: Porsche Taycan launch**
- Porsche платит за 3 месяца показов своей новой модели
- В tutorial показывается Taycan → наши диски на нём
- Win-win: Porsche получает premium-аудиторию, мы — доход + PR

**Сценарий 2: Ротация брендов**
- Клиент из Германии → Mercedes AMG в tutorial
- Клиент из UK → Aston Martin
- Клиент из Italy → Maserati
- Клиент из US → Corvette C8

**Сценарий 3: Trade deal (старт)**
- На ранней стадии — бесплатное размещение в обмен на:
  - Упоминание в пресс-релизах автобренда
  - Социальное продвижение
  - Доступ к их пресс-фото для маркетинга

### Техническая реализация

**MVP:**
- 1-3 tutorial-видео с разными машинами
- Ротация случайная или по гео
- Формат: 15-30 sec, mp4, low-weight для быстрой загрузки
- Плейсмент: hero section, popup при первом визите, или onboarding

**Продвинутая версия:**
- A/B тест какие машины конвертят лучше
- Персонализация под интересы клиента
- Dynamic ad insertion на основе поведения

### Почему это работает

| Фактор | Почему |
|---|---|
| **Дополнительный доход** | Монетизируем каждый визит без продажи дисков |
| **Повышение статуса бренда** | Ассоциация с Porsche/AMG = premium positioning |
| **SEO/PR эффект** | Автобренды линкуются на нас, взаимная реклама |
| **Уже платный user-flow** | Пользователь всё равно смотрит tutorial — почему не монетизировать |
| **Natural fit** | Машины + диски = органичная связка, не навязчивая реклама |

### Риски и как нивелировать

| Риск | Решение |
|---|---|

## 🔗 ЭКОСИСТЕМА: Интеграция с проектом Психавто

**Идея владельца:** объединить два его проекта в единую экосистему cross-sell.

### Что такое Психавто

**Psihavto.tilda.ws** — сервис психологического подбора автомобиля. Клиент проходит психологический тест → получает рекомендацию какой автомобиль ему подходит → переход на сайт выбора авто.

### Логика интеграции

```
Психавто: «Вам подходит BMW M3»
         ↓
Клиент купил/выбрал авто
         ↓
Forged: «А теперь сделайте его СВОИМ»
         ↓
Подбор дисков под конкретную модель
         ↓
Продажа дисков
```

**Идея:** Психавто продаёт автомобили (партнёрство) → Forged продаёт диски на этот же автомобиль → два дохода с одного клиента.

### Варианты интеграции (для обсуждения)

#### Вариант 1: Cross-sell funnel
**Одна воронка, две точки монетизации**
- В конце теста Психавто → CTA "Now personalize it" → redirect на Forged
- Forged знает какую машину выбрал клиент → предзаполнение модели для примерки
- Единый customer journey: тест → авто → диски

#### Вариант 2: Психологический подбор дисков
**Используем ту же философию**
- У Psihavto: "Какой ВАШ автомобиль" (психологический тест)
- У Forged: "Какие ВАШИ диски" (упрощённый тест на основе той же методологии)
- Одна ДНК бренда = "мы знаем вас лучше чем вы сами"

#### Вариант 3: Shared customer base
**Общая клиентская база**
- Клиент Психавто → 10% скидка в Forged
- Клиент Forged → ссылка на Психавто если ему интересно
- Совместная email-рассылка

#### Вариант 4: Widget интеграция
**Взаимные виджеты**
- На Психавто виджет Forged: "Already chose a car? Customize its wheels"
- На Forged виджет Психавто: "Not sure which car? Take the test"

#### Вариант 5: Единый бренд-зонтик
**Один parent-бренд, два продукта**
- Например: "Auto Sense" — parent brand
  - "Sense" (Психавто) — найти свою машину
  - "Forged" — сделать её своей
- Единый checkout, один аккаунт
- Больше cross-sell возможностей

### Потенциальная синергия

| Психавто даёт Forged | Forged даёт Психавто |
|---|---|
| Горячих лидов с выбранным авто | Дополнительная value после теста |
| Психологический профиль клиента | Визуальный AI-tool для контента |
| Русскоязычную аудиторию | Глобальный охват |
| SEO-трафик по темам "подбор авто" | Статусность (premium бренд) |
| Email-базу | Monetization опыт |

### Важные нюансы

⚠️ **Разные рынки:**
- Психавто = Россия (Tilda, русский)
- Forged = мир (EN/DE/ES/etc., без России)

**Не смешивать языки и рынки!** Интеграция возможна, но с сегментацией:
- Для русскоязычных клиентов Психавто → русская версия Forged (возможно отдельная)
- Для международных клиентов Forged → опционально упоминание методологии из Психавто
